# Machine Translator

Machine Translator for multiple languages using LangChain

Supported Languages:

- English
- Spanish
- Italian
- French

## Setup

Developed with

- Python 3.12
- Pip installer
- MacOS 14.5, M2 chip

Install Python packages (optionally inside a virtual environment)
`pip install -r requirements.txt`

## Running

Use the `LangChainApp` from `langchain_app.py`
